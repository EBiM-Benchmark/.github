# EBiM — Embodied Bimanual Manipulation Competition

Workshop competition at **IROS 2026** on embodied bimanual manipulation with the Mobile FR3 Duo platform.

- Workshop site: <https://iros-ebim.github.io/>
- IROS 2026: September 27 – October 1, 2026

## What's in this org

| Repo | Status | Purpose |
|---|---|---|
| [iros-ebim.github.io](https://github.com/IROS-EBiM/iros-ebim.github.io) | Public | Workshop website |
| benchmark | Coming soon | Simulation benchmark environment (Isaac Sim / MuJoCo) |
| baselines | Coming soon | Reference baseline implementations |
| evaluation | Private | Scoring infrastructure (held-out) |
| leaderboard | Coming soon | Public leaderboard frontend |
| robot-control | Private | ROS 2 packages for Mobile FR3 Duo |
| dataset-tools | Coming soon | Data conversion and validation |
