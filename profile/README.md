# EBiM — Embodied Bimanual Manipulation Competition

Workshop competition at **IROS 2026** on embodied bimanual manipulation with the Mobile FR3 Duo platform.

- Workshop site: <https://ebim-benchmark.github.io/>
- IROS 2026: September 27 – October 1, 2026

> **Status:** The benchmark, baselines, and tooling are in active scaffolding; first public release targets v1.0 alongside IROS 2026.

## What's in this org

| Repo | Visibility | Purpose |
|---|---|---|
| [ebim-benchmark.github.io](https://github.com/EBiM-Benchmark/ebim-benchmark.github.io) | Public | Workshop website |
